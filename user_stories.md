# Board Game Community Hub – User Stories

## 1. User Registration & Authentication

- As a new user, I want to sign up with my name, email, and password, so that I can create an account and access the app.
- As a returning user, I want to log in with my credentials, so that I can access my games and scores.
- As a user, I want to log out, so that my account remains secure.

## 2. Dashboard

- As user wants to view all my scores for different games on my dashboard, so that I can track my performance.\*\*
- As user wants to see a list of all games I have participated in, so that I can quickly access them.\*\*
- As user I want to click on my highest score in a game and see all other scores I’ve submitted in other games, so that I can compare my performance across games.\*\*

## 3. Managing Games

- As user wants to add a new game with its name and genre, so that I can track it in the hub.\*\*
- As user wants to view a game’s details and see all scores submitted by other users, so that I can compare performances.\*\*
- As a user want to click a user’s highest score in a game and view all of that user’s scores in other games, so that I can track other players’ progress.\*\*

## 4. Submitting Scores

- As user wants to submit a score for a specific game, including points and optionally a round number or date, so that my profile and leaderboard are updated.\*\*

## 5. Validation & Rules

- A user want to see all fields to be required unless specified, so that my data is complete.\*\*
- A user want the app to prevent duplicate emails, so that accounts are unique.\*\*
- As a user I want to see numeric validation for scores, so that invalid data is not accepted.\*\*
- As a user I want validation errors to appear inline in forms, so that I know exactly what needs fixing.\*\*

## 6. Navigation

- As a user I want to use the top menu to switch between Dashboard and Games, so that I can easily navigate the app.\*\*
- As a user I want forms to be straightforward and show errors if something is missing, so that I can submit data correctly.\*\*
